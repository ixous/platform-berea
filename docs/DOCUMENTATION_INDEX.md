\# Bienvenido

Este documento sirve como índice principal de toda la documentación técnica del proyecto.

Antes de comenzar cualquier desarrollo, todos los integrantes del equipo (humanos o asistentes de IA) deberán leer esta documentación en el orden indicado.

La documentación representa la fuente oficial de verdad del proyecto.

\---

\# Filosofía

Este proyecto sigue un principio muy simple:

\*\*Arquitectura primero. Código después.\*\*

No se desarrollará ninguna funcionalidad que no esté previamente documentada.
