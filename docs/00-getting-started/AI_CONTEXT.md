\# Descripción del Proyecto

Centro Cristiano Berea Website es el sitio web institucional oficial de Centro Cristiano Berea, ubicado en Mexicali, Baja California, México.

El objetivo del proyecto es desarrollar una plataforma moderna, rápida, segura y completamente administrable que represente dignamente la identidad de la iglesia y permita mantener actualizado su contenido sin depender de un desarrollador.

Este proyecto no es un ERP ni un sistema empresarial.

Es un sitio web institucional con un CMS moderno.
